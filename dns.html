üöÄ Initial Raspberry Pi Setup

1 Built case and installed NVME

Prepared the Raspberry Pi 5 Ensured the Raspberry Pi 5 was free from thermal pads, heatsinks, or HATs. Inserted the CR1220 battery into the RTC holder. Connected the PCIe and RTC battery cables, noting the orientation of the FPC PCIe cable.

2 Installed the Power Button

Mounted the power button in the case and connect it to the Raspberry Pi 5 using the provided cables.

3 Attached the Tower Cooler

Applied thermal pads to the BCM2712 SoC, wireless module, and VLI chip on the Raspberry Pi 5. Installed the tower cooler using the provided plastic bits with springs. Connected the 4-pin fan wire to the fan connector on the Raspberry Pi 5.

4 Installed the M.2 SSD

I used a NVMe SSD, installed the stud in the appropriate location 2242. Secured the SSD in the M.2 M key slot and fastened it with screws. Connected the SSD to the PCIe adapter board.

5 Assembled the Case

Installed the two fans with dust filters into the case. Attached the acrylic plate to the IO HAT+ after removing protective films. Connected the fans and OLED display. Inserted the IO HAT+ into the 40-pin GPIO header of the Raspberry Pi 5. Secured both metal parts of the case together. Installed the remaining acrylic plates and secured them with screws.

üï≥Ô∏è Pi-hole Installation & Configuration

1 Installed Pi-hole

Ran the automated installer:

curl -sSL https://install.pi-hole.net | bash

During installation, configure:

  • Upstream DNS servers (Cloudflare: 1.1.1.1, 1.0.0.1)
  • Block lists (select default lists)
  • Admin web interface (Yes)
  • Web server (Yes)
  • Query logging (Yes)
  • Privacy mode (Show everything)
Note: Save the admin password displayed at the end of installation!

2 Configured Router DNS Settings

Update your router's DNS settings to point to your Pi-hole server:

  • Primary DNS: 192.168.1.100 (your Pi-hole IP)
  • Secondary DNS: 1.1.1.1 (backup)

üîß Pi-hole Configuration Files

Main config: /etc/pihole/pihole-FTL.conf

Custom DNS: /etc/pihole/custom.list

Whitelist: /etc/pihole/whitelist.txt

Blacklist: /etc/pihole/blacklist.txt

3 Accessed Web Interface

navigated to:

http://192.168.1.100/admin

Logged in with the password from installation and explored the dashboard.
