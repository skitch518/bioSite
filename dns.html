Key Features

  • Network-wide Ad Blocking: Block ads and trackers across all devices
  • Custom DNS Records: Easy access to local services (e.g., server.local)
  • High Availability: Automatic failover to backup DNS servers
  • Detailed Analytics: Monitor DNS queries and blocked content
  • Low Power Consumption: Runs 24/7 on minimal electricity

üçìPi-OS Installation & Configuration

1 Installed Pi-OS on Micro SD

Downloaded the installer from Raspberry Pi Software. You can purchase a Raspberry Pi with a preformatted card, but I recommend formatting it yourself to have more control, such as setting passwords and enabling SSH, among other configurations.

2 Downloaded Pi-Hole

I used Claude to walk me through the installation of Pi-Hole. I am running the server headless, but I initially failed to enable SSH correctly. As a result, I had to connect a monitor to the Pi to properly connect it to my main PC.

3 Connected Client PC to DNS

I continued the setup process with Claude as my guide. I use Spectrum as my ISP, and my router blocks me from making adjustments, so I had to manually set up each client.

4 Set Up NVMe as Boot Drive

After successfully connecting my PC as a client to the new DNS server, I realized I had forgotten to set up the 1TB NVMe drive. Once again, Claude helped me partition the drive, install the necessary software, and configure it to boot from the drive instead of the microSD card. This setup provides increased storage and faster read/write speeds.

What I Would Do Differently

I used AI to assist me with this project, but for future projects, I'd like to take more notes and learn more from the process. I also want to try using an Orange Pi. Although they are not as well-supported, they offer much greater performance while still being affordable.
